# 메뉴판 번역 서비스 (Menu Translator)

## 📝 프로젝트 소개
메뉴판 번역 서비스는 외국인 관광객들과 해외 여행객들을 위한 웹 서비스입니다. 메뉴판 사진을 업로드하면 ChatGPT의 Vision 기능을 활용하여 메뉴를 인식하고 원하는 언어로 번역해주는 서비스입니다.

## 🚀 주요 기능
- 메뉴판 이미지 업로드
- ChatGPT Vision을 활용한 메뉴 인식
- 다국어 번역 제공
- 직관적인 사용자 인터페이스

## 🛠 기술 스택
- Frontend: React
- AI: ChatGPT API (Vision)
